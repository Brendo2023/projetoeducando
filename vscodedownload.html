Visual Studio Code


Visual studio code foi desenvolvido pela microsoft. Sim! A empresa do Tio Gades. No dia 29 de Abril de 2015, essa maravilhosa ferramenta foi lançada em sua primeira versão. Desde lá, vem tendo melhorias constantes. Aliás! Esse artigo foi escrito nele.

Um dos recursos do VScode, para os mais íntimos, é o InteliSense. Quando se pria qualquer código de qualquer linguagem a qual a suporte pelo VScode, o próprio programa completa sua sintaxe. Diminuindo assim, o tempo de desenvolvimento.

Outra característica notável do VScode, é o fato dele possuir um servidor próprio. Esse servidor roda em uma porta local, criando um local host que é atualizado automáticamente.

O usuário também pode conectar seu site ao Microsoft Azure e hospedar sua aplicação na Web a partir do próprio VScode.

VScode também conta com suporte a várias linguagens de programação, incluindo: Java, Python, C e C++, JavaScript, C# e HTML com CSS.
